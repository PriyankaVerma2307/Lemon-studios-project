## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/PriyankaVerma2307/Lemon-studios-project
```

Go to project folder:

```bash
cd Lemon-project
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
